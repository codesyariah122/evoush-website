### Migration Server to Heroku  

Configure server on heroku : 

##### .user.ini (Custome for php at server)  
```bash
upload_max_filesize=100M
post_max_size=100M
```

##### .htaccess  

```bash
<IfModule mod_rewrite.c>
RewriteEngine on
RewriteRule ^$ public/ [L]
RewriteRule (.*) public/$1 [L]
</IfModule>
```
