backend : <a href="https://app.evoush.com">Evoush Backend</a>

* Front End
    - WebPack
    - Bootstrap v4.6
    - Vue JS
    - Blade Template
    - jQuery

* Backend 
    - Laravel 8 as a Framework
    - Backend
    - RestApi
